# Node.js Quiz App

A simple quiz application built with Node.js to test knowledge about Node.js.

## Installation

1. Make sure you have Node.js installed.
2. Clone or download this project.
3. Run `npm install` (though no dependencies are needed for this simple app).

## Running the App

Run `npm start` or `node app.js` to start the quiz.

The app will ask you a series of questions about Node.js. Answer them in the console.

## Questions

The quiz includes questions about:
- What is Node.js?
- How to install Node.js
- What is npm?
- Creating a new Node.js project
- Purpose of package.json

Enjoy!
